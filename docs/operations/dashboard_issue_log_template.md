# Dashboard Issue Log Template

Copy this template for daily dashboard operating notes.

```text
date:
run_id:
dashboard_page:
issue_type:
observed_behavior:
expected_behavior:
data_file_involved:
severity:
follow_up_needed:
owner:
status:
notes:
```

Suggested issue types:

```text
missing_data
stale_data
layout
confusing_label
source_coverage
history
browser_console
workflow
other
```

Severity labels:

```text
low
medium
high
blocker
```

Keep issue notes focused on dashboard operation and data display. Scoring
formula changes should be handled in a separate evidence-backed milestone.
